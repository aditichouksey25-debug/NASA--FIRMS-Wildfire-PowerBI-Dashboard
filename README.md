PROJECT OVERVIEW -

This project presents an interactive Power BI dashboard that analyzes Nasa FIRMS wildfire data to study fire occurences in Madhya Pradesh, India. It uses geospatial mapping and time-based analysis to identify fire hotspots, seasonal trends, and high-risk regions.

OBJECTIVE-

To visualize and analyze wildfire patterns in Madhya Pradesh using satellite-based NASA FIRMS data and derive meaningful insights for environmental monitoring and decision-making.

DATA SOURCE-

NASA FIRMS (Fire Information for Resource Management System).

TOOLS USED-

Microsoft Power BI.
Microsoft Excel (Data cleaning and preprocessing).
NASA FIRMS dataset.

FEATURES OF DASHBOARD-

Geospatial mapping of fire locations across Madhya Pradesh.
Monthly wildfire trent analysis.
Interactive slicers (month,day/night,fire intensity).
Hotspot identification based on fire density.
Visual breakdown of fire occurences over time.

KEY INSIGHTS-

Certain districts show consistently higher fire activity(hotspots).
Fire incidents peak during specific dry months.
Clear spatial clustering of fire events in southern region.
Seasonal variation strongly impacts fire frequency.
